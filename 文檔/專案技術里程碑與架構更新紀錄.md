# 專案技術里程碑與架構更新紀錄

本文件記錄了本專案（MR60-FLIR-FHIR 系統）在技術架構優化與部署階段的重要里程碑與更新細節。

---

## 📌 技術里程碑

### 1. 資料庫與多執行緒伺服器雙邊上線 ✅
*   **更新細節**：
    *   本機與學校電腦的地端 **PostgreSQL** 已全部配置完成，資料庫與資料表 (`test_temperature`) 成功建立。
    *   後端 Python 伺服器 (`server.py`) 升級為 **`ThreadingHTTPServer` 多執行緒架構**。
    *   在資料庫連線層中加入 `try...finally` 資源回收機制，徹底解決了原本因為 HTTP Keep-Alive 長連接與資料庫連線洩漏造成的伺服器卡死轉圈問題，實現秒級讀寫響應。

### 2. 成功註冊專屬網域與永久命名隧道綁定 ✅
*   **更新細節**：
    *   成功註冊專屬網域 **`60gigahertz.uk`**。
    *   在 Cloudflare 中建立了名為 **`school-fhir`** 的永久命名隧道 (Named Tunnel)。
    *   將該網域的 CNAME 指向永久隧道，不再需要使用每次啟動時隨機產生、且容易造成 DNS 快取延遲的 `trycloudflare.com` 臨時快速通道。
    *   公網測試網址永久固定為 **`https://60gigahertz.uk`**，網頁與 API 通訊具備受信任的 SSL 加密證書。

### 3. 完成官方 FHIR 實作規格與存儲機制文檔 ✅
*   **更新細節**：
    *   撰寫並提交了 `衛生福利部FHIR與FHIR_Server定義說明.md` 與 `HAPI_FHIR_JPA_資料格式與存儲機制說明.md`。
    *   內容詳解了 HL7 FHIR 國際標準、衛生福利部台灣核心實作指引 (TW Core IG) 的在地化對接、以及 HAPI FHIR JPA 底層使用 PostgreSQL 進行 JSON 壓縮與索引（Search Parameters）的儲存原理，作為本專案的核心學術與專案申報依據。

## 🔐 學校電腦安全部署指南 (無須登入 Cloudflare 帳號)

為了保障您 Cloudflare 個人帳戶的安全，**請勿在學校電腦執行 `cloudflared tunnel login`**，這會產生 `cert.pem` 系統證書，進而暴露您的帳號管理權限。

請採用以下任一無毒、安全的部署方案：

### 方案 A：僅複製「單一隧道憑證 JSON」（最推薦）
1.  **在家裡/本機電腦**，打開以下路徑：
    `C:\Users\Typelin_Station\.cloudflared\`
2.  複製這一個 JSON 檔案：
    **`4e76d50f-490d-4bce-9ecb-0fd1c0b5707a.json`**（這是 `school-fhir` 的專屬隧道金鑰）。
3.  **不要複製 `cert.pem`**！
4.  將該 JSON 檔案傳到學校電腦的對應路徑：
    `C:\Users\[學校電腦使用者]\.cloudflared\4e76d50f-490d-4bce-9ecb-0fd1c0b5707a.json`
5.  在學校電腦執行啟動指令：
    ```cmd
    C:\Users\Typelin_Station\Desktop\cloudflared.exe tunnel run --url http://localhost:8080 school-fhir
    ```

> 🔒 **安全性說明**：此 JSON 檔案只包含啟動 `school-fhir` 隧道的連線金鑰，**完全不包含您帳號的密碼或管理權限**。學校電腦只能「運行」這個隧道，無法修改您的 DNS 設定，也無法建立或刪除您的其他網域（例如 `.me`），極度安全。

### 方案 B：使用 Token 啟動（學校電腦不需要任何憑證檔案）
1.  在本機電腦登入您的 Cloudflare Zero Trust 網頁後台：
    進入 **`Access` ➔ `Tunnels`**，找到 `school-fhir`，點擊右側的 **`Configure`**。
2.  在 **Install connector** 頁面中，複製那一串很長的 **`Token`** 碼。
3.  在學校電腦的 CMD 中，直接輸入此指令啟動（完全不需要在 `.cloudflared` 放任何檔案）：
    ```cmd
    C:\Users\Typelin_Station\Desktop\cloudflared.exe tunnel run --token <您複製的Token金鑰>
    ```
